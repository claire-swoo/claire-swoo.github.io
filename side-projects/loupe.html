Chrome Extension

Loupe

AI-powered design and clarity critique for any webpage — judges pages on their own terms and returns structured, actionable feedback.

Add to Chrome GitHub →
What it does

Loupe opens as a side panel and analyzes the current page — detecting its type (portfolio, STEM paper, landing page, etc.) and evaluating it on criteria appropriate to that context. Each issue gets a severity label and a before / after suggestion.

Context-aware analysis
Detects page type and switches evaluation criteria accordingly — a research portfolio isn't judged like a marketing site.
Severity badges
Issues are tagged Critical / Major / Minor so you know what to fix first.
Before / after cards
Every suggestion shows the current state and a concrete rewrite, not just vague advice.
Four themes
Navy, Charcoal, Sky, and Paper White — switch without reloading.
Draggable panel
Minimize to a strip or drag it out of the way while you edit.
arXiv HTML support
Analyzes arXiv papers via their HTML rendering — PDF canvas is inaccessible by design.

Install

Chrome Web Store link coming soon. To install from source in the meantime:

  1. Clone or download the repo
  2. Go to chrome://extensions and enable Developer Mode
  3. Click Load unpacked and select the extension folder
  4. Open any webpage, click the Loupe icon in the toolbar

Background

I was building a personal site and wanted feedback that understood what the page was trying to do — not generic SEO tips. Existing tools either check accessibility rules mechanically or give off-target AI suggestions with no context about the page type. Loupe started as a quick experiment to fix that.
